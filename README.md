# 🌤️ Weather App

A real-time weather app built with Next.js and OpenWeatherMap API.

## Features
- Live weather data for any city in the world
- Animated effects (rain, snow, sun rays) based on weather
- Secure API key hidden with server-side routes
- Deployed on Vercel

## Built With
- Next.js
- JavaScript
- OpenWeatherMap API
- Vercel
